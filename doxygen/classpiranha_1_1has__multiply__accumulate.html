template<typename T, typename U = T, typename V = U>
class piranha::has_multiply_accumulate< T, U, V >

Type trait to detect the availability of piranha::math::multiply_accumulate.

This type trait will be true if piranha::math::multiply_accumulate can be called with arguments of type T, U and V, false otherwise.
