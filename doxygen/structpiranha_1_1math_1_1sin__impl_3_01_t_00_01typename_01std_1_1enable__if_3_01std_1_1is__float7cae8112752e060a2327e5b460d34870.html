Detailed Description

template<typename T>
struct piranha::math::sin_impl< T, typename std::enable_if< std::is_floating_point< T >::value >::type >

Specialisation of the piranha::math::sin() functor for floating-point types.

This specialisation is activated when T is a C++ floating-point type. The result will be computed via the standard std::sin() function.
