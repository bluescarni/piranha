Detailed Description

template<typename Series>
struct piranha::math::t_ldegree_impl< Series, typename std::enable_if< is_instance_of< Series, trigonometric_series >::value >::type >

Specialisation of the piranha::math::t_ldegree() functor for instances of piranha::trigonometric_series.

This specialisation is activated if Series is an instance of piranha::trigonometric_series. The corrsponding method in piranha::trigonometric_series will be used for the computation, if available.
