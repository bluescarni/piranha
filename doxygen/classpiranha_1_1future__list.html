Detailed Description

template<typename F>
class piranha::future_list< F >

Class to store a list of futures.

This class is a minimal thin wrapper around an std::list of std::future objects. The class provides convenience methods to interact with the set of futures in an exception-safe manner.

Type requirements

F must be an instance of std::future.
