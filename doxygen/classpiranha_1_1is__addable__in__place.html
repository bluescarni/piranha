Detailed Description

template<typename T, typename U = T>
class piranha::is_addable_in_place< T, U >

In-place addable type trait.

Will be true if objects of type U can be added in-place to objects of type T.

This type trait will strip T and U of reference qualifiers, and it will test the operator in the form

operator+=(Td &, const Ud &)

where Td and Ud are T and U after the removal of reference qualifiers. E.g.:

is_addable_in_place<int>::value == true;
is_addable_in_place<int,std::string>::value == false;
