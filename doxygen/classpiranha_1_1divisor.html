Detailed Description

template<typename T>
class piranha::divisor< T >

Divisor class.

This class is used to represent keys of the form

where are integers, are symbols, and are positive integers. The type of and is T. The terms of the product are stored in a piranha::hash_set and they are guaranteed to be in a canonical form defined by the following properties:

  • if T is a C++ integral type, the values of and are within implementation-defined ranges,
  • is always strictly positive,
  • the first nonzero in each term is positive,
  • the in each term have no non-unitary common divisor.

Type requirements

T must be either a C++ signed integral type or an instance of piranha::mp_integer.

Exception safety guarantee

Unless otherwise specified, this class provides the strong exception safety guarantee for all operations.

Move semantics

Move semantics is equivalent to the move semantics of piranha::hash_set.

Serialization

This class supports serialization.
