Detailed Description

Symbol set.

This class represents an ordered set of piranha::symbol. The individual piranha::symbol instances can be accessed via iterators or the index operator.

Exception safety guarantee

This class provides the strong exception safety guarantee for all operations.

Move semantics

Move construction and move assignment will leave the moved-from object in a state equivalent to a default-constructed object.

Serialization

This class supports serialization.
