Detailed Description

template<typename To, typename From>
class piranha::key_is_convertible< To, From >

Detect if a key type is convertible to another key type.

This type trait will be true if the key type To can be constructed from a const reference to the key type From and a const reference to piranha::symbol_set, false otherwise.

The decay types of To and From are considered by this type trait. To and From must satisfy piranha::is_key, otherwise a compile-time error will be generated.
