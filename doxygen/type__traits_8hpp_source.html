421 // at best across GCC/Clang/Intel. The only one to get it completely right at this time seems
422 // GCC 4.9.0. The others:
423 // - GCC < 4.9.0 compiles this but it has problems when TT has a certain number of template arguments > 0

425 // class <typename T, typename U, typename ... Args> class TT;

800 // Detect the availability of std::iterator_traits on type It, plus a couple more requisites from the
