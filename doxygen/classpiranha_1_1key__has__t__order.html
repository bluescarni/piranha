Detailed Description

template<typename Key>
class piranha::key_has_t_order< Key >

Type trait to detect if a key type has a trigonometric order property.

The type trait has the same meaning as piranha::has_t_order, but it's meant for use with key types. It will test the presence of two t_order() const methods, the first one accepting a const instance of piranha::symbol_set, the second one a const instance of piranha::symbol_set::positions and a const instance of piranha::symbol_set.

Key must satisfy piranha::is_key.
